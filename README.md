This are scripts used for analysis of ex-vivo electrophysiology data

Master file all_peak_results_final.xlsx contains filepaths, genotypes, cell IDs, condition (baseline, Noradrenaline, wash), comment (exclusion etc), 
and peak results. This is the file to which resting membrane potential analysis is appended. 

For resting membrane potential the latest script including majority of the preprocessing, analysis and plotting is rmp_plotting_by_genotypes_final.ipynb
other related scripts include: rmp_from_databsae.ipynb, ephys_rmp_plotting.py, ephys_rmp_create_csv.py
